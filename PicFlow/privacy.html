Privacy Policy

Last updated: March 23, 2026

🛡️ PicFlow does not collect, store, or transmit any personal data to third-party servers. Your privacy is our top priority.

Overview

PicFlow (hereinafter referred to as "the App") is a macOS menu bar image hosting tool that helps users upload images to cloud storage platforms (GitHub, Alibaba Cloud OSS, Tencent Cloud COS) and obtain image links. This privacy policy explains how the App handles your information.

Data Collection

The App does not collect any personal information.

Specifically:

  • Does not collect user identity information (name, email, device identifiers, etc.)
  • Does not collect usage analytics or behavioral data
  • Does not use any third-party analytics or tracking SDKs
  • Contains no ads or ad tracking
  • Does not use cookies or similar tracking technologies

Locally Stored Data

The App stores the following information locally on your device. This data is stored only on your macOS system and is never uploaded to any server:

Data Type Purpose Storage Method
Cloud storage keys (API Token / AccessKey / SecretKey) Connect to your cloud storage service macOS Keychain (encrypted)
Bucket configuration (Bucket, Region, domain, etc.) Specify the target location for image uploads SwiftData local database
Hotkey settings Store user-defined global hotkeys macOS UserDefaults
Upload history Display links and thumbnails of uploaded images SwiftData local database
Image compression settings Store user's image compression preferences macOS UserDefaults
Language preference Store user's chosen interface language macOS UserDefaults

Network Communication

The App only communicates with the cloud storage services you configure:

  • GitHub API（api.github.com）— When GitHub is selected as the image host
  • Alibaba Cloud OSS（*.oss-*.aliyuncs.com）— When Alibaba Cloud OSS is selected as the image host
  • Tencent Cloud COS（*.cos.*.myqcloud.com）— When Tencent Cloud COS is selected as the image host

All network requests use HTTPS encryption. The App does not communicate with any other servers or third-party services.

💡 Your images are uploaded directly from your device to your own cloud storage service, without passing through any third-party proxy or relay server.

Clipboard & Screenshot Access

The App accesses the system clipboard and screenshots in the following situations:

  • When you press the upload hotkey, it reads images from the clipboard
  • When auto screenshot detection is enabled, it monitors system screenshot events
  • When you open the menu bar popover, it shows a preview of clipboard images

The App does not send clipboard content to any third-party server. The screenshot detection feature can be disabled in settings at any time.

Accessibility Permission

The App requires macOS Accessibility permission to register global hotkeys. This permission is only used to listen for keyboard shortcut events and is not used for any other purpose.

Third-Party Services

The App uses the official APIs of the following third-party cloud storage services to upload images:

  • GitHub REST API
  • Alibaba Cloud Object Storage Service (OSS)
  • Tencent Cloud Object Storage (COS)

When you upload images using the App, the relevant data is subject to the privacy policies and terms of service of the respective platforms.

Other than these, the App does not integrate any third-party SDKs or services.

Children's Privacy

The App is not intended for children under 13. The App does not knowingly collect any personal information from children.

Policy Changes

If there are significant changes to this privacy policy, we will notify you through app updates or this page. We recommend checking this page periodically for the latest privacy policy.
