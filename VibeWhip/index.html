为 Vibe Coder 而生

一站式管理你的 AI 编程工作流，从 Prompt 到交付全程掌控

📚

Prompt 库管理

按 6 大类别（项目前、周期性、项目后、技巧型 Prompt、技巧技能、工具箱）管理你的 Prompt。支持图片粘贴、一键复制、复制即发送，工具箱类别可存储 URL 并自动显示域名。告别混乱的提示词管理。

📋

项目追踪

为每个 Vibe Coding 项目创建独立面板。颜色标签（6 种预设 + 自定义）、项目目录关联、归档/恢复、Focus 标记、Markdown 导出、拖拽排序。

🗒️

屏幕贴纸

浮动 NSPanel 贴纸：8 种预设 + 自定义颜色，WCAG 智能字体配色。支持置顶、拖拽、缩放、位置记忆，批量显示/隐藏/清除。

🔀

IDE 切换器

浮动面板一键切换 Cursor、Windsurf、VS Code、Xcode 等 13+ 内置 IDE，还可添加自定义 IDE。面板支持边缘吸附和 Peek 模式。

🔥

Whip Board 看板

为 Focus 标记的项目打造的看板视图。FlowLayout 自动换行，卡片宽度/高度可调，拖拽重排，每张卡片可直接输入 Todo。配合 Vibing 模式的发光效果，让你全身心投入编码。

🟣

Vibing 模式

为每个项目开启 Vibing 状态（⌘⇧P）。紫色动画横幅、侧边栏脉冲圆点、Whip Board 发光特效——沉浸心流编码。

💭

Think Notes

每个项目配备独立的富文本编辑器，自动保存思考笔记。随时记录灵感、决策和关键上下文，无需离开工作台。

☁️

iCloud 同步

自动检测 iCloud 可用性，透明切换本地/iCloud。JSON 持久化带防抖优化，多台 Mac 无缝衔接。

🌍

40 种语言

内置 40 种语言/地区支持（含跟随系统选项），自动跟随系统语言或手动切换。7 页引导流程助你快速上手。
