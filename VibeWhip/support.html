🚀 Quick Start

1

Install VibeWhip

Download and install VibeWhip from the Mac App Store. A 7-page onboarding guide will appear on first launch to help you quickly learn the core features.

2

Build Your Prompt Library

Add your frequently used prompts to the Prompt Library. Categorize them across 6 categories (Before Project, Periodic, After Project, Trick Prompt, Trick Skill, Toolbox). Supports image paste, one-click copy, and copy-on-send. The Toolbox category can store URLs directly.

3

Create Projects & Manage Todos

Click "New Project" in the sidebar or use ⌘N to create a project. Set color tags, link project directories, track progress with Todos (image paste, Shift+Enter for line-break submit), and capture thinking with Think Notes.

4

Boost: Stickers + IDE Switch + Vibing Mode

Create screen stickers via menu bar or hotkeys. Use the IDE Switcher to quickly switch between editors. Activate Vibing mode (⌘⇧P) for flow state, and visually manage Focus projects on the Whip Board.

❓ FAQ

How to customize hotkeys?

+
ℹ Feature

Open the main window, click the settings button (⚙️) in the toolbar, then find the "Hotkeys" section on the settings page. You can customize hotkeys for:

  • Create new screen sticker
  • Show/hide all stickers
  • Clear all stickers
  • IDE Switcher

Click the hotkey field, then press your desired key combination to complete the setup.

Screen stickers disappeared?

+
⚠ Usage

Stickers may be in a "hidden" state. You can restore them by:

  1. Click the VibeWhip icon in the menu bar → "Screen Stickers" → "Show All Stickers"
  2. Or use the hotkey (check default in Settings) to toggle sticker visibility
  3. Sticker data will automatically restore after restarting the app

IDE switcher can't detect my editor?

+
⚠ Permission

The IDE Switcher requires Accessibility permission to enumerate and switch windows. Please go to:

System Settings → Privacy & Security → Accessibility

Find VibeWhip in the list and enable it. If already enabled but still not working, uncheck and recheck it, then restart the app.

Where is data stored? Will it be lost?

+
🔒 Privacy

All VibeWhip data (projects, prompts, stickers, Think Notes, settings, etc.) is stored on your local device. iCloud sync to your other Macs is supported, with automatic data migration between local and iCloud storage.

Data is never uploaded to any third-party servers and is completely under your control. We recommend regular Time Machine backups as a precaution.

How to switch app language?

+
ℹ Feature

VibeWhip supports 40 language/region variants (including a "Follow System" option). It follows the system language by default, or you can manually switch in Settings:

Settings → Language

Select your desired language from the dropdown. A restart of the app is required for the change to fully take effect.

Is there a limit on Prompt Library?

+
ℹ Feature

No limit. You can add as many prompts as you like, organized freely across 6 categories: pre-project, periodic, post-project, tips prompts, tips skills, and toolbox. Each prompt supports one-click copy to clipboard.

What is Whip Board?

+
ℹ Feature

Whip Board is a Kanban view designed for Focus-marked projects. Use ⌘⇧F to quickly open it. Cards support FlowLayout auto-wrapping, adjustable width/height, drag reorder, and per-card Todo input. Combined with Vibing mode glow effects, it helps you stay focused during Vibe Coding.

What is Vibing Mode? How to use it?

+
ℹ Feature

Vibing Mode is VibeWhip's immersive coding experience. Use ⌘⇧P to toggle the Vibing state for the current project. When activated, you'll see:

  • A purple animated banner at the top of the project detail page
  • A pulse dot next to the project in the sidebar
  • Glow effects on the project card in Whip Board

Each project can have Vibing toggled independently, helping you mark which projects you're fully committed to.

What is Think Notes? How to use it?

+
ℹ Feature

Think Notes is a rich text editor available for each project. Click the "Think Notes" tab in the project detail page to open it. Content auto-saves, no manual action needed. Perfect for capturing ideas, architecture decisions, and key context.

How to archive projects? How to export Markdown?

+
ℹ Feature

Both project archiving and Markdown export can be found in the project action menu:

  • Archive: Right-click a project or use the project menu → "Archive". Archived projects can be restored from the archive list
  • Export Markdown: Select "Export as Markdown" from the project menu to export project info, todo list, etc. as a .md file

Can I paste images in Todos and Prompts?

+
ℹ Feature

Yes! When adding a Todo or Prompt, you can paste images directly from the clipboard (⌘V). Images will be embedded in the content, making it easy to record screenshots, designs, or reference materials. Todos also support click-to-copy and Shift+Enter for line-break submit.

iCloud sync not working?

+
⚠ Usage

VibeWhip auto-detects iCloud availability and seamlessly switches between local/iCloud storage. If sync issues occur, please check:

  1. Make sure you're signed in to your Apple account and iCloud Drive is enabled in System Settings
  2. Check System Settings → Apple ID → iCloud → iCloud Drive → Ensure VibeWhip is enabled
  3. Ensure your network connection is stable
  4. Try restarting the app — VibeWhip re-detects iCloud on launch and auto-migrates data
