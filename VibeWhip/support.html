🚀 快速开始

1

安装 VibeWhip

从 Mac App Store 下载安装 VibeWhip，首次启动时会显示引导页面，帮助你了解核心功能。

2

构建你的 Prompt 库

在首页的 Prompt 库中添加你常用的 Prompt，按类别分类管理（项目前、周期性、项目后、技巧型等）。支持一键复制，随时调用。

3

创建项目

点击侧边栏的「新建项目」或使用快捷键 ⌘N，为你的每个 Vibe Coding 任务创建独立的项目面板，追踪进度和管理 Todo。

4

使用屏幕贴纸和 IDE 切换器

通过菜单栏或快捷键创建屏幕贴纸，在编码时随时记录想法。使用 IDE 切换器在多个编辑器之间快速切换，提升工作效率。

❓ 常见问题

如何自定义快捷键？

+
ℹ 功能相关

打开主窗口，点击工具栏的设置按钮（⚙️），进入设置页面后找到「快捷键」区域。你可以为以下功能自定义快捷键：

  • 新建屏幕贴纸
  • 显示/隐藏所有贴纸
  • 清除所有贴纸
  • IDE 切换器

点击快捷键区域，然后按下你想要的组合键即可完成设置。

屏幕贴纸消失了怎么办？

+
⚠ 使用相关

贴纸可能处于「隐藏」状态。你可以通过以下方式恢复：

  1. 点击菜单栏的 VibeWhip 图标 → 「屏幕贴纸」→「显示所有贴纸」
  2. 或使用快捷键（默认可在设置中查看）来切换贴纸的显示/隐藏
  3. 贴纸数据会在重启后自动恢复

IDE 切换器检测不到我的编辑器？

+
⚠ 权限相关

IDE 切换器需要「辅助功能」权限来枚举和切换窗口。请前往：

系统设置 → 隐私与安全性 → 辅助功能

在列表中找到 VibeWhip 并启用。如果已启用但仍不生效，可以取消勾选后重新勾选，然后重启 App。

数据存储在哪里？会丢失吗？

+
🔒 隐私相关

VibeWhip 的所有数据（项目、Prompt、贴纸、设置等）都存储在你的本地设备上。支持 iCloud 同步到你的其他 Mac。

数据不会上传到任何第三方服务器，完全由你掌控。建议定期使用 Time Machine 备份以防万一。

如何切换 App 界面语言？

+
ℹ 功能相关

VibeWhip 支持 39 种语言。默认跟随系统语言，也可以在设置中手动切换：

设置 → 语言

从下拉菜单中选择你想要的语言，切换后需要重启 App 才能完全生效。

Prompt 库有数量限制吗？

+
ℹ 功能相关

没有限制。你可以添加任意数量的 Prompt，按 6 个类别自由分类管理：项目前、周期性、项目后、技巧型 Prompt、技巧技能、工具箱。每个 Prompt 支持一键复制到剪贴板。

WhipBoard 暴走模式是什么？

+
ℹ 功能相关

WhipBoard 是一个灵感看板功能，快捷键 ⌘⇧F 可快速打开。你可以把待办、想法、灵感快速扔进来，以卡片形式视觉化整理，帮助你在 Vibe Coding 过程中保持思路清晰。
