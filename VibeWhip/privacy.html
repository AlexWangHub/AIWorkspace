隐私政策

最后更新日期：2026 年 3 月 23 日

🛡️ VibeWhip 不收集、不存储、不传输任何用户个人数据到第三方服务器。你的隐私是我们的首要考量。

概述

VibeWhip（以下简称「本应用」）是一款 macOS 项目管理工具，专为 Vibe Coding 工作流设计，提供 Prompt 库管理、项目追踪、屏幕贴纸、IDE 切换器等功能。本隐私政策说明了本应用如何处理你的信息。

数据收集

本应用不收集任何个人信息。

具体而言：

  • 不收集用户身份信息（姓名、邮箱、设备标识符等）
  • 不收集使用分析数据或行为数据
  • 不使用任何第三方分析或追踪 SDK
  • 不包含广告或广告追踪
  • 不使用 Cookie 或类似追踪技术

本地存储的数据

本应用会在你的设备本地存储以下信息，这些数据仅保存在你的 macOS 系统中，不会上传到任何第三方服务器：

数据类型 用途 存储方式
Prompt 库内容 存储用户创建的 Prompt 标题和内容 本地文件（Application Support）
项目数据 存储项目名称、Todo 列表、进度状态 本地文件（Application Support）
屏幕贴纸 存储贴纸内容、位置、大小、置顶状态 本地文件（Application Support）
快捷键设置 记录用户自定义的全局快捷键 macOS UserDefaults
语言偏好 记录用户选择的界面语言 macOS UserDefaults
应用设置 Todo 行为偏好、引导页状态等 macOS UserDefaults

iCloud 同步

本应用支持通过 Apple iCloud 在你的多台 Mac 之间同步数据。

  • 同步的数据包括：Prompt 库、项目数据、屏幕贴纸等
  • 数据通过 Apple 的 iCloud 基础设施进行传输和存储
  • iCloud 同步受到 Apple 隐私政策的保护
  • 本应用不会在 iCloud 之外的任何第三方服务器上存储或处理你的数据

💡 iCloud 同步完全由 Apple 管理，你的数据在传输和存储过程中均受到 Apple 的端到端加密保护。

网络通信

本应用不会主动连接任何外部服务器。唯一的网络通信发生在启用 iCloud 同步时，由 macOS 系统自动完成，本应用不直接参与网络请求。

辅助功能权限

本应用需要 macOS 辅助功能（Accessibility）权限来实现以下功能：

  • IDE 切换器：枚举并切换正在运行的 IDE 窗口
  • 全局快捷键：注册和监听全局键盘快捷键

该权限仅用于上述功能，不用于其他任何目的。本应用不会通过辅助功能权限读取或记录你的键盘输入。

第三方服务

本应用不集成任何第三方 SDK 或服务。

不使用 Firebase、Google Analytics、Facebook SDK 或任何其他第三方追踪、分析或广告服务。

儿童隐私

本应用不针对 13 岁以下的儿童。本应用不会有意收集任何儿童的个人信息。

隐私政策变更

如本隐私政策有重大变更，我们将通过应用更新或本页面进行通知。建议你定期查看本页面以了解最新的隐私政策。
