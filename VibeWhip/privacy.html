Privacy Policy

Last updated: March 24, 2026

🛡️ VibeWhip does not collect, store, or transmit any personal user data to third-party servers. Your privacy is our top priority.

Overview

VibeWhip (hereinafter "the App") is a macOS project management tool designed for the Vibe Coding workflow, offering Prompt Library management, project tracking, screen stickers, IDE switcher, Whip Board, Vibing mode, Think Notes, and iCloud sync. This privacy policy explains how the App handles your information.

Data Collection

This App does not collect any personal information.

Specifically:

  • No collection of user identity information (name, email, device identifiers, etc.)
  • No collection of usage analytics or behavioral data
  • No third-party analytics or tracking SDKs
  • No ads or ad tracking
  • No cookies or similar tracking technologies

Locally Stored Data

The App stores the following information locally on your device. This data is only saved on your macOS system and is not uploaded to any third-party servers:

DATA TYPE PURPOSE STORAGE METHOD
Prompt Library Content Stores user-created prompt titles, content, and images (6 categories) JSON files (Application Support / iCloud)
Project Data Stores project names, color tags, todo lists (with images), archive status, Focus marking, Vibing state, directory paths JSON files (Application Support / iCloud)
Think Notes Stores per-project rich text thinking notes (auto-saved) JSON files (Application Support / iCloud)
Screen Stickers Stores sticker content, position, size, color, pin status JSON files (Application Support / iCloud)
Hotkey Settings Records 4 user-defined global hotkeys macOS UserDefaults
Language Preference Records user-selected interface language (40 options) macOS UserDefaults
App Settings Todo behavior preferences, onboarding status, window positions, etc. macOS UserDefaults

iCloud Sync

The App supports syncing data between your multiple Macs via Apple iCloud.

  • Synced data includes: Prompt Library, project data, screen stickers, Think Notes, etc.
  • Data is transmitted and stored via Apple's iCloud infrastructure
  • The App auto-detects iCloud availability and transparently switches and migrates between local and iCloud storage
  • iCloud sync is protected by Apple's privacy policy
  • The App does not store or process your data on any third-party servers outside of iCloud

💡 iCloud sync is fully managed by Apple. Your data is protected by Apple's end-to-end encryption during both transmission and storage.

Network Communication

The App does not actively connect to any external servers. The only network communication occurs when iCloud sync is enabled, handled automatically by macOS. The App does not directly participate in network requests.

Accessibility Permission

The App requires macOS Accessibility permission to enable the following features:

  • IDE Switcher: enumerate and switch between running IDE windows via AXUIElement public API
  • Global Hotkeys: register and listen for global keyboard shortcuts

This permission is used solely for the above features and no other purpose. The App does not read or record your keyboard input via the Accessibility permission.

Apple Events Permission

The App requests Apple Events (Automation) permission to enable IDE switching functionality. This permission allows the App to send window activation commands to other apps. The App only uses this permission when the user actively triggers an IDE switch, and does not automatically execute or monitor any other app behavior.

System API Declarations

Per Apple's PrivacyInfo requirements, the App declares the use of the following system APIs:

  • UserDefaults (NSPrivacyAccessedAPICategoryUserDefaults): used for storing app settings, hotkey configurations, and user preferences. Reason code: CA92.1
  • File Timestamp (NSPrivacyAccessedAPICategoryFileTimestamp): used for file timestamp management during data persistence. Reason code: C617.1

The App declares NSPrivacyTracking as false and does not perform any user tracking.

Third-Party Services

This App does not integrate any third-party SDKs or services.

No Firebase, Google Analytics, Facebook SDK, or any other third-party tracking, analytics, or advertising services are used.

Children's Privacy

This App is not directed at children under 13. The App does not intentionally collect any personal information from children.

Policy Changes

If significant changes are made to this privacy policy, we will notify you via app updates or this page. We recommend checking this page periodically for the latest privacy policy.
