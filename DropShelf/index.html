🖱️

Multiple Trigger Methods

Auto-appears when dragging files, screen edge hot zone activation, global hotkey (⌥Space), menu bar icon click — four trigger methods cover all scenarios, open the shelf anytime, anywhere.

📚

Multi-Shelf Management

Create multiple shelves, organize by project or task. Switch tabs to freely operate between different shelves.

💾

Persistent Storage

Stored files won't be lost after restart. All shelf contents are automatically persisted, your data remains intact on next boot.

🗜️

ZIP Compression

Select multiple files and compress to ZIP archive with one click, no extra tools needed, with real-time progress.

🖼️

Image Conversion & PDF Merge

Batch convert image formats (PNG, JPEG, HEIC, WebP, TIFF), merge multiple images/PDFs with one click. Powerful file operations make the shelf not just storage, but a lightweight workstation.

👁️

Quick Look Preview

Press Space to quickly preview files in the shelf, supporting images, PDFs, documents and more.

✏️

Batch Rename

Select files and rename directly, with batch support for more efficient file organization.

📌

Menu Bar Resident

Quietly stays in the menu bar, no Dock space occupied. Pure native implementation, lightweight and always ready on startup.
