隐私政策

最后更新日期：2026 年 3 月 23 日

🛡️ DropShelf 是一款完全离线运行的本地工具，不收集、不存储、不传输任何用户数据到远程服务器。你的文件永远留在你的 Mac 上。

概述

DropShelf（以下简称「本应用」）是一款 macOS 原生文件搁置架工具，帮助用户在拖拽文件时将文件暂存到浮动面板中进行管理和操作。本隐私政策说明了本应用如何处理你的信息。

数据收集

本应用不收集任何个人信息。

具体而言：

  • 不收集用户身份信息（姓名、邮箱、设备标识符等）
  • 不收集使用分析数据或行为数据
  • 不使用任何第三方分析或追踪 SDK
  • 不包含广告或广告追踪
  • 不使用 Cookie 或类似追踪技术

网络通信

本应用不进行任何网络通信。

DropShelf 是一款完全离线运行的工具，不连接任何远程服务器，不发送任何网络请求。所有文件操作（暂存、压缩、转换、重命名、合并）均在你的 Mac 本地完成。

💡 DropShelf 不需要互联网连接即可正常使用。你的文件内容永远不会通过网络传输到任何地方。

本地存储的数据

本应用会在你的设备本地存储以下信息，这些数据仅保存在你的 macOS 系统中：

数据类型 用途 存储方式
搁置架内容 持久化暂存的文件引用，重启后恢复 本地 JSON 文件（Application Support）
文件书签 Security-Scoped Bookmark，跨会话访问文件 本地 JSON 文件（Application Support）
快捷键设置 记录用户自定义的全局快捷键 macOS UserDefaults
面板位置 记住浮动面板的位置和大小 macOS UserDefaults
语言偏好 记录用户选择的界面语言 macOS UserDefaults

文件访问

本应用会读取和操作你拖入搁置架的文件，仅在以下情况下发生：

  • 当你将文件拖入搁置架时（创建文件引用）
  • 当你对文件执行操作时（压缩、转换、重命名、合并等）
  • 当你预览搁置架中的文件时（Quick Look）
  • 当你将文件从搁置架拖出到目标位置时

本应用不会在后台扫描你的文件系统，也不会访问搁置架之外的文件。

辅助功能权限

本应用需要 macOS 辅助功能（Accessibility）权限来实现以下功能：

  • 全局快捷键 — 注册和响应全局键盘快捷键（默认 ⌥Space）
  • 拖拽监听 — 检测系统级文件拖拽事件以自动弹出搁置架面板

该权限不用于其他任何目的，不会监控键盘输入或屏幕内容。

第三方服务

本应用不集成任何第三方 SDK、框架或服务。所有功能均使用 macOS 原生框架（AppKit、SwiftUI、PDFKit、ImageIO 等）实现。

儿童隐私

本应用不针对 13 岁以下的儿童。本应用不会有意收集任何儿童的个人信息。

隐私政策变更

如本隐私政策有重大变更，我们将通过应用更新或本页面进行通知。建议你定期查看本页面以了解最新的隐私政策。
