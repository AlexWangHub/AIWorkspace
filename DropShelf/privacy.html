Privacy Policy

Last updated: March 23, 2026

🛡️ DropShelf is a fully offline local tool that does not collect, store, or transmit any user data to remote servers. Your files always stay on your Mac.

Overview

DropShelf (hereinafter referred to as "the App") is a native macOS file shelf tool that helps users temporarily store files on a floating panel for management and operations while dragging. This privacy policy explains how the App handles your information.

Data Collection

The App does not collect any personal information.

Specifically:

  • No collection of user identity information (name, email, device identifiers, etc.)
  • No collection of usage analytics or behavioral data
  • No use of any third-party analytics or tracking SDKs
  • No advertisements or ad tracking
  • No use of cookies or similar tracking technologies

Network Communication

The App does not perform any network communication.

DropShelf is a fully offline tool that does not connect to any remote servers or send any network requests. All file operations (storing, compression, conversion, renaming, merging) are performed locally on your Mac.

💡 DropShelf does not require an internet connection to function. Your file contents will never be transmitted over the network to anywhere.

Locally Stored Data

The App stores the following information locally on your device. This data is only saved on your macOS system:

DATA TYPE PURPOSE STORAGE METHOD
Shelf Contents Persist stored file references, restore after restart Local JSON file (Application Support)
File Bookmarks Security-Scoped Bookmark, cross-session file access Local JSON file (Application Support)
Hotkey Settings Store user-customized global hotkeys macOS UserDefaults
Panel Position Remember floating panel position and size macOS UserDefaults
Language Preference Store user-selected interface language macOS UserDefaults

File Access

The App reads and operates on files you drag into the shelf, only in the following situations:

  • When you drag files into the shelf (creating file references)
  • When you perform operations on files (compression, conversion, renaming, merging, etc.)
  • When you preview files in the shelf (Quick Look)
  • When you drag files from the shelf to a target location

The App does not scan your file system in the background, nor access files outside the shelf.

Accessibility Permission

The App requires macOS Accessibility permission for the following features:

  • Global Hotkey — Register and respond to global keyboard shortcuts (default ⌥Space)
  • Drag Monitoring — Detect system-level file drag events to auto-show the shelf panel

This permission is not used for any other purpose, and does not monitor keyboard input or screen content.

Third-Party Services

The App does not integrate any third-party SDKs, frameworks, or services. All features are implemented using native macOS frameworks (AppKit, SwiftUI, PDFKit, ImageIO, etc.).

Children's Privacy

The App is not directed at children under 13. The App does not intentionally collect any personal information from children.

Policy Changes

If there are significant changes to this privacy policy, we will notify you through app updates or this page. We recommend checking this page regularly for the latest privacy policy.
