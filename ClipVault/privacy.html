Privacy Policy

Last updated: March 23, 2026

🛡️ ClipVault is a 100% offline app. It does not collect, store, or transmit any user data to any server. Your privacy is our top priority.

Overview

ClipVault (hereinafter referred to as "this app") is a macOS smart clipboard manager that helps users auto-capture and manage clipboard history, with built-in area screenshot and screen recording features. This privacy policy explains how this app handles your information.

Data Collection

This app does not collect any personal information.

Specifically:

  • No collection of user identity information (name, email, device identifiers, etc.)
  • No collection of usage analytics or behavioral data
  • No use of any third-party analytics or tracking SDKs
  • No advertisements or ad tracking
  • No use of cookies or similar tracking technologies

Network Communication

This app does not establish any network connections.

ClipVault is a fully offline app that does not communicate with any server. No API calls, no cloud sync, no telemetry reporting. All features run entirely on your local Mac.

💡 ClipVault does not require an internet connection to work. Your clipboard data, screenshots, and recordings are always stored locally on your Mac and never leave your device.

Locally Stored Data

This app stores the following information locally on your device. This data is only saved on your macOS system and will not be uploaded to any server:

DATA TYPE PURPOSE STORAGE METHOD
Clipboard History Save text, links, images, etc. copied by the user Local SwiftData Database
Image Raw Data Save original binary data of copied images or screenshots Local SwiftData Database
Screenshot Files PNG images generated by area screenshot feature Local file (default: Desktop, customizable)
Recording Files MOV videos generated by screen recording feature Local file (default: Movies folder, customizable)
Hotkey Settings Store user-defined three global hotkeys macOS UserDefaults
User Preferences Store interface language, history limit, cleanup policy, etc. macOS UserDefaults
Custom Tags Category tags added by the user to clipboard records Local SwiftData Database

Clipboard Access

The core function of this app is to monitor and manage system clipboard content. When the "Auto-capture Clipboard" feature is enabled, the app checks the system clipboard for new content at 1.5-second intervals and saves new content to the local database.

You can disable the auto-capture feature at any time in settings. Once disabled, this app will not read the clipboard.

Screen Recording Permission

The area screenshot and screen recording features require the macOS "Screen Recording" permission. This permission is used only for the following purposes:

  • Area Screenshot: Uses the CGWindowListCreateImage API to capture the user-selected screen area
  • Screen Recording: Uses the AVCaptureScreenInput API to record the user-selected screen area or fullscreen

Captured or recorded content is saved locally on your Mac only and will not be transmitted to any server. You can revoke this permission at any time in System Settings.

OCR Text Recognition

This app uses the Apple Vision framework to perform local OCR text recognition on images. The entire recognition process runs on your device, and image data is never sent to any server.

Launch at Login

This app can optionally be set as a login item using the macOS SMAppService API for launch at login. This feature only takes effect when you actively enable it and can be disabled at any time in settings.

Data Deletion

You have full control over your data:

  • Delete history records individually
  • Clear all history data with one click in settings
  • Manually delete screenshot and recording files
  • Clean up specified storage directories with one click in settings
  • After uninstalling, delete the ~/Library/Application Support/ClipVault/ directory to remove all data

Third-Party Services

This app does not integrate any third-party SDKs or services. No ads, no analytics tools, no crash reporting platforms, no social media SDKs. The app is built entirely with Apple native frameworks.

Children's Privacy

This app is not directed at children under 13. This app does not intentionally collect any personal information from children.

Policy Changes

If there are significant changes to this privacy policy, we will notify you through app updates or this page. We recommend checking this page periodically for the latest privacy policy.
