隐私政策

最后更新日期：2026 年 3 月 23 日

🛡️ ClipVault 是一款 100% 离线应用。不收集、不存储、不传输任何用户数据到任何服务器。你的隐私是我们的首要考量。

概述

ClipVault（以下简称「本应用」）是一款 macOS 智能剪贴板管理器，帮助用户自动捕获和管理剪贴板历史，并提供区域截图与屏幕录制功能。本隐私政策说明了本应用如何处理你的信息。

数据收集

本应用不收集任何个人信息。

具体而言：

  • 不收集用户身份信息（姓名、邮箱、设备标识符等）
  • 不收集使用分析数据或行为数据
  • 不使用任何第三方分析或追踪 SDK
  • 不包含广告或广告追踪
  • 不使用 Cookie 或类似追踪技术

网络通信

本应用不建立任何网络连接。

ClipVault 是一款完全离线的应用，不与任何服务器通信。没有 API 调用、没有云同步、没有遥测数据上报。应用的所有功能均在你的 Mac 本地完成。

💡 ClipVault 不需要互联网连接即可正常工作。你的剪贴板数据、截图和录屏文件始终存储在你的 Mac 本地，从不离开你的设备。

本地存储的数据

本应用会在你的设备本地存储以下信息，这些数据仅保存在你的 macOS 系统中，不会上传到任何服务器：

数据类型 用途 存储方式
剪贴板历史记录 保存用户复制的文本、链接、图片等内容 本地 SwiftData 数据库
图片原始数据 保存复制的图片或截图的原始二进制数据 本地 SwiftData 数据库
截图文件 区域截图功能生成的 PNG 图片 本地文件（默认桌面，可自定义）
录屏文件 屏幕录制功能生成的 MOV 视频 本地文件（默认影片文件夹，可自定义）
快捷键配置 记录用户自定义的三个全局快捷键 macOS UserDefaults
用户偏好设置 记录界面语言、历史上限、清理策略等设置 macOS UserDefaults
自定义标签 用户为剪贴板记录添加的分类标签 本地 SwiftData 数据库

剪贴板访问

本应用的核心功能是监控和管理系统剪贴板内容。当「自动捕获剪贴板」功能开启时，本应用会以 1.5 秒的间隔检查系统剪贴板是否有新内容，并将新内容保存到本地数据库。

你可以随时在设置中关闭自动捕获功能。关闭后，本应用不会读取剪贴板。

屏幕录制权限

本应用的区域截图和屏幕录制功能需要 macOS「屏幕录制」权限。该权限仅用于以下目的：

  • 区域截图：使用 CGWindowListCreateImage API 截取用户选定的屏幕区域
  • 屏幕录制：使用 AVCaptureScreenInput API 录制用户选定的屏幕区域或全屏

截取或录制的内容仅保存在你的 Mac 本地，不会传输到任何服务器。你可以随时在系统设置中撤销该权限。

OCR 文字识别

本应用使用 Apple Vision 框架在本地对图片进行 OCR 文字识别。整个识别过程完全在你的设备上进行，图片数据不会被发送到任何服务器。

开机自启

本应用可选设置为登录项，使用 macOS SMAppService API 实现开机自启。该功能仅在你主动开启时生效，且可以随时在设置中关闭。

数据删除

你对自己的数据拥有完全控制权：

  • 可以逐条删除历史记录
  • 可以在设置中一键清除所有历史数据
  • 可以手动删除截图和录屏文件
  • 可以在设置中一键清理指定存储目录
  • 卸载本应用后，删除 ~/Library/Application Support/ClipVault/ 目录即可清除所有数据

第三方服务

本应用不集成任何第三方 SDK 或服务。没有广告、没有分析工具、没有崩溃报告平台、没有社交媒体 SDK。应用完全使用 Apple 原生框架构建。

儿童隐私

本应用不针对 13 岁以下的儿童。本应用不会有意收集任何儿童的个人信息。

隐私政策变更

如本隐私政策有重大变更，我们将通过应用更新或本页面进行通知。建议你定期查看本页面以了解最新的隐私政策。
