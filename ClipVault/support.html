🚀 Quick Start

1

Install and Launch ClipVault

Download ClipVault from the Mac App Store. On first launch, follow the setup guide to complete basic configuration.

2

Grant Screen Recording Permission

To use screenshot or recording features, the system will prompt you to grant "Screen Recording" permission. Go to System Settings → Privacy & Security → Screen Recording, find ClipVault and enable it.

3

Configure Global Hotkeys (Optional)

Set global hotkeys for three functions in settings: Open Panel (suggested ⌘⇧V), Area Screenshot (suggested ⌘⇧S), Screen Recording (suggested ⌘⇧R).

4

Start Using

Copy anything as usual, ClipVault will auto-capture and save. Click the menu bar icon or press the hotkey to view history, search content, and one-click copy back to clipboard.

❓ FAQ

Screenshot or recording not working

+
⚠ Permission

Screenshot and screen recording require the "Screen Recording" permission. Please go to:

System Settings → Privacy & Security → Screen Recording

Find ClipVault in the list and enable it. If already enabled but still not working, uncheck and recheck it, then restart ClipVault.

Global hotkey not responding

+
⚠ Configuration

Please check the following:

  1. Confirm you have configured hotkeys in settings (no default hotkeys on initial install)
  2. Ensure hotkeys don't conflict with other apps
  3. Hotkeys must include at least one modifier key (⌘ / ⌃ / ⌥ / ⇧)
  4. Try resetting the hotkey: clear first, then re-record

Clipboard not auto-captured

+
⚠ Feature

Please confirm the following:

  1. In Settings → General, ensure the "Auto-capture Clipboard" toggle is on
  2. ClipVault needs to be running in the background (icon visible in menu bar)
  3. If history reaches the limit (default 500), the oldest records will be auto-cleaned

How to change screenshot/recording save location?

+
ℹ Feature

You can customize the save location for three types of data:

Settings → Storage Location

  • Clipboard Records: Database storage directory
  • Screenshot Files: Default save to Desktop, customizable
  • Recording Files: Default save to ClipVault Recordings in Movies folder, customizable

Click the "Change Path" button to select a new storage directory.

How to set up auto-cleanup?

+
ℹ Feature

Go to Settings → History, where you can:

  • Set maximum saved entries (100 / 200 / 500 / 1000 / 2000)
  • Enable auto-cleanup and set retention days

Old records exceeding the limit will be auto-cleaned on next launch. Pinned records are not affected by auto-cleanup.

How to switch app language?

+
ℹ Feature

ClipVault supports 39 languages. It follows the system language by default, but you can switch manually:

Settings → General → Interface Language

After selecting the target language, restart the app for full effect.

Where is OCR text recognition?

+
ℹ Feature

Click any image-type record to enter the detail page, where you'll see the "OCR Recognition" button. ClipVault uses the Apple Vision framework to locally recognize text in images, and results can be copied with one click.

OCR processing is done entirely locally — no images are uploaded to any server.

Does the app upload data to servers?

+
🔒 Privacy

No. ClipVault is a 100% offline app that does not establish any network connections. All clipboard data, screenshots, and recordings are stored locally on your Mac. No cloud sync, no analytics tracking, no data uploads. See our privacy policy for details.
