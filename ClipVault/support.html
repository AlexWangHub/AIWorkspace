🚀 快速开始

1

安装并启动 ClipVault

从 Mac App Store 下载 ClipVault。首次启动时，按照引导页的指引完成基本设置。

2

授予屏幕录制权限

若要使用截图或录屏功能，系统会提示你授予「屏幕录制」权限。前往 系统设置 → 隐私与安全性 → 屏幕录制，找到 ClipVault 并启用。

3

配置全局快捷键（可选）

在设置中为三个功能设定全局快捷键：呼出面板（建议 ⌘⇧V）、区域截图（建议 ⌘⇧S）、屏幕录制（建议 ⌘⇧R）。

4

开始使用

正常复制任何内容，ClipVault 会自动捕获并保存。点击菜单栏图标或按下快捷键即可查看历史、搜索内容、一键复制回剪贴板。

❓ 常见问题

截图或录屏功能无法使用

+
⚠ 权限相关

截图和屏幕录制功能需要「屏幕录制」权限。请前往：

系统设置 → 隐私与安全性 → 屏幕录制

在列表中找到 ClipVault 并启用。如果已启用但仍不生效，可以取消勾选后重新勾选，然后重启 ClipVault。

全局快捷键没有反应

+
⚠ 配置相关

请检查以下几点：

  1. 确认你已经在设置中配置了快捷键（初始安装时没有默认快捷键）
  2. 确保快捷键没有与其他应用冲突
  3. 快捷键必须包含至少一个修饰键（⌘ / ⌃ / ⌥ / ⇧）
  4. 尝试重新设置快捷键：先清除，再重新录入

剪贴板内容没有被自动捕获

+
⚠ 功能相关

请确认以下几点：

  1. 在设置 → 通用中，确保「自动捕获剪贴板」开关已开启
  2. ClipVault 需要在后台运行（菜单栏中有图标）
  3. 如果历史记录达到上限（默认 500 条），最旧的记录会被自动清理

如何更改截图和录屏的存储位置？

+
ℹ 功能相关

你可以分别自定义三种数据的存储位置：

设置 → 存储位置

  • 剪贴板记录：数据库存储目录
  • 截图文件：默认保存到桌面，可自定义
  • 录屏文件：默认保存到「影片」文件夹下的 ClipVault Recordings，可自定义

点击「修改路径」按钮选择新的存储目录。

如何设置历史记录自动清理？

+
ℹ 功能相关

前往 设置 → 历史记录，你可以：

  • 设置最大保存条数（100 / 200 / 500 / 1000 / 2000）
  • 开启自动清理，设置保留天数

超出限制的旧记录会在下次启动时自动清理。置顶的记录不受自动清理影响。

如何切换 App 界面语言？

+
ℹ 功能相关

ClipVault 支持 39 种语言。默认跟随系统语言，也可以手动切换：

设置 → 通用 → 界面语言

选择目标语言后，需要重启 App 才能完全生效。

图片的 OCR 文字识别在哪里？

+
ℹ 功能相关

点击任何图片类型的记录进入详情页，你会看到「OCR 识别」按钮。点击后，ClipVault 使用 Apple Vision 框架在本地识别图片中的文字，识别结果可以一键复制。

OCR 处理完全在本地进行，不会上传图片到任何服务器。

App 会上传我的数据到服务器吗？

+
🔒 隐私相关

不会。ClipVault 是一款 100% 离线应用，不建立任何网络连接。所有剪贴板数据、截图、录屏文件均存储在你的 Mac 本地。没有云同步、没有分析追踪、没有数据上传。详情请参阅我们的隐私政策。
