隐私政策

最后更新日期：2026 年 3 月 22 日

🛡️ ClipToNotion 不收集、不存储、不传输任何用户个人数据到第三方服务器。你的隐私是我们的首要考量。

概述

ClipToNotion（以下简称「本应用」）是一款 macOS 菜单栏工具，帮助用户将剪贴板文字内容发送到 Notion 页面。本隐私政策说明了本应用如何处理你的信息。

数据收集

本应用不收集任何个人信息。

具体而言：

  • 不收集用户身份信息（姓名、邮箱、设备标识符等）
  • 不收集使用分析数据或行为数据
  • 不使用任何第三方分析或追踪 SDK
  • 不包含广告或广告追踪
  • 不使用 Cookie 或类似追踪技术

本地存储的数据

本应用会在你的设备本地存储以下信息，这些数据仅保存在你的 macOS 系统中，不会上传到任何服务器：

数据类型 用途 存储方式
Notion API Token 连接你的 Notion 账户 macOS UserDefaults
Page / Database ID 指定内容发送的目标页面 macOS UserDefaults
快捷键设置 记录用户自定义的全局快捷键 macOS UserDefaults
上传记录 显示历史发送内容的摘要 本地文件（Application Support）
语言偏好 记录用户选择的界面语言 macOS UserDefaults

网络通信

本应用仅与以下服务通信：

  • Notion 官方 API（api.notion.com）— 用于发送剪贴板内容和测试连接

所有网络请求均使用 HTTPS 加密传输。本应用不会与任何其他服务器或第三方服务通信。

💡 你的剪贴板内容从你的设备直接发送到 Notion 服务器，中间不经过任何第三方代理或中转服务器。

剪贴板访问

本应用会读取系统剪贴板的文字内容，仅在以下情况下发生：

  • 当你按下发送快捷键时
  • 当你打开剪贴板面板预览内容时

本应用不会在后台持续监控剪贴板，也不会记录剪贴板历史。

辅助功能权限

本应用需要 macOS 辅助功能（Accessibility）权限来注册全局快捷键。该权限仅用于监听键盘快捷键事件，不用于其他任何目的。

第三方服务

本应用使用 Notion 的官方公开 API 来发送内容。当你使用本应用发送内容到 Notion 时，相关数据将受到 Notion 的隐私政策约束。

除此之外，本应用不集成任何第三方 SDK 或服务。

儿童隐私

本应用不针对 13 岁以下的儿童。本应用不会有意收集任何儿童的个人信息。

隐私政策变更

如本隐私政策有重大变更，我们将通过应用更新或本页面进行通知。建议你定期查看本页面以了解最新的隐私政策。
