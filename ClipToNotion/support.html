🚀 快速开始

1

创建 Notion Integration

前往 Notion Integrations 页面，点击「New integration」创建一个新的 Integration，并复制 API Token（以 ntn_ 或 secret_ 开头）。

2

关联目标页面

在 Notion 中打开你想要发送内容的目标页面，点击页面右上角的 ··· 菜单 → 「Connections」→ 添加刚创建的 Integration。

3

配置 ClipToNotion

打开 ClipToNotion 设置界面，粘贴 API Token 和目标页面 URL（或 Page ID），点击「测试连接」确认连接成功。

4

开始使用

复制任意文字内容，按下全局快捷键（默认 ⌘⇧V），剪贴板内容即刻发送到 Notion！

❓ 常见问题

连接测试失败，提示「TLS 安全连接失败」

+
⚠ 网络相关

这通常是网络代理或 VPN 干扰了与 Notion 服务器的加密通信。请尝试：

  1. 切换代理节点，或开启全局代理模式
  2. 暂时关闭代理/VPN 后重试
  3. 如果使用公司网络，可能防火墙阻止了对 api.notion.com 的访问

连接测试失败，提示「无效的 API Token」

+
⚠ 配置相关

请检查以下几点：

  1. 确认 Token 以 ntn_ 或 secret_ 开头
  2. 确保复制时没有多余的空格
  3. 前往 Notion Integrations 确认 Token 未被撤销

快捷键无法触发

+
⚠ 权限相关

全局快捷键需要「辅助功能」权限。请前往：

系统设置 → 隐私与安全性 → 辅助功能

在列表中找到 ClipToNotion 并启用。如果已启用但仍不生效，可以取消勾选后重新勾选。

提示「无效的 Page ID」怎么办？

+
⚠ 配置相关

ClipToNotion 支持直接粘贴 Notion 页面的完整 URL，例如：

https://www.notion.so/Your-Page-Title-abc123def456

也支持直接填写 32 位的 Page ID。请确保：

  1. 目标页面已经关联了你的 Integration
  2. URL 或 ID 复制完整，没有遗漏字符

发送成功但 Notion 页面看不到内容？

+
⚠ 使用相关

内容默认追加到页面顶部。请尝试：

  1. 刷新 Notion 页面（按 ⌘R）
  2. 滚动到页面顶部查看
  3. 确认你查看的页面与设置中配置的 Page ID 一致

如何切换 App 界面语言？

+
ℹ 功能相关

ClipToNotion 支持 39 种语言。默认跟随系统语言，也可以在设置中手动切换：

设置 → 语言设置 → 界面语言

切换后部分界面需要重新打开才能生效。

App 是否会上传我的数据到第三方服务器？

+
🔒 隐私相关

不会。ClipToNotion 仅在你的设备本地运行，所有剪贴板数据通过 Notion 官方 API 直接发送到你的 Notion 账户，全程不经过任何第三方服务器。API Token 仅存储在你的 macOS 系统中。
