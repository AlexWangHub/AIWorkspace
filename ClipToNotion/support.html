🚀 Quick Start

1

Create Notion Integration

Go to the Notion Integrations page, click "New integration", create a new Integration, and copy the API Token (starting with ntn_ or secret_).

2

Connect Target Page

Open your target Notion page, click the ··· menu → "Connections" → add the Integration you just created.

3

Configure ClipToNotion

Open ClipToNotion settings, paste the API Token and target page URL (or Page ID), click "Test Connection" to confirm.

4

Start Using

Copy any text, press the global hotkey (default ⌘⇧V), and clipboard content is instantly sent to Notion!

❓ FAQ

Connection test failed: "TLS secure connection failed"

+
⚠ Network

This is usually caused by a proxy or VPN interfering with encrypted communication to Notion. Try:

  1. Switch proxy nodes or enable global proxy mode
  2. Temporarily disable proxy/VPN and retry
  3. Corporate firewalls may block access to api.notion.com

Connection test failed: "Invalid API Token"

+
⚠ Configuration

Please check the following:

  1. Confirm Token starts with ntn_ or secret_
  2. Ensure no extra spaces when copying
  3. Check Notion Integrations to confirm Token is not revoked

Hotkey not working

+
⚠ Permissions

Global hotkeys require Accessibility permission. Go to:

System Settings → Privacy & Security → Accessibility

Find ClipToNotion in the list and enable it. If already enabled but not working, uncheck and recheck.

What if it says "Invalid Page ID"?

+
⚠ Configuration

ClipToNotion supports pasting the full Notion page URL, e.g.:

https://www.notion.so/Your-Page-Title-abc123def456

You can also enter the 32-character Page ID. Make sure:

  1. The target page has your Integration connected
  2. URL or ID is fully copied without missing characters

Sent successfully but can't see content in Notion?

+
⚠ Usage

Content is appended to the top of the page by default. Try:

  1. Refresh the Notion page (press ⌘R)
  2. Scroll to the top of the page
  3. Confirm the page matches the configured Page ID

How to switch app language?

+
ℹ Feature

ClipToNotion supports 39 languages. It follows system language by default, or you can switch manually:

Settings → Language → Interface Language

Some interfaces need to be reopened after switching.

Does the app upload data to third-party servers?

+
🔒 Privacy

No. ClipToNotion runs locally on your device. All clipboard data is sent directly to your Notion account via the official API, never through third-party servers. API Token is stored only on your macOS system.
